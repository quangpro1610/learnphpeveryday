# Day 01 - Tính tổng $n + $nn + $nnn
This repository to Learn PHP everyday with my friend @lyquochuy1988

### Mô tả

Cho trước số nguyên dương **n**(n < 10). Bạn hãy viết chương trình để in ra tổng có dạng

- n + nn + nnn

| Input | Output |
| ---   | ---    |
| n = 1 | 1 + 11 + 111 = 123 |
| n = 2 | 2 + 22 + 222 = 246 |
| n = 3 | 3 + 33 + 333 = 369 |
| n = 4 | 4 + 44 + 444 = 492 |
| n = 5 | 5 + 55 + 555 = 615 |
